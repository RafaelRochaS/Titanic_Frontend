This project has the objective of displaying MLOps end-to-end capacities, taking the traditional 'Titanic survivors' problem and showcasing a full application around it, complete with a trained model on the backend, and capabilities to receive user input in the frontend, returning whether or not the user would have survived the crash, based on the inputs.

In the future, the application will also display Machine Learning metrics and backend stats, available on a separate endpoint.
